# TheVipersCove

## This project is made with;

The code is written in:
- Typescript
- ES6
- UTF-8

Base:
- MongoDB
- Angular 7+
- Express
- Nodejs
Also known as MEAN stack

Middleware:
- Mongoose
- Nodemon
- Protractor
- Jsonwebtoken
- Concurrently
- Cors
- PassportJS
- PassportJWT
- bcryptjs
- body-parser

This app also includes:
- Karma
- Jasmine
But this is not used yet.
